## 一个轻量版的仿koa框架

### 框架源码目录：src/

### 如何运行demo

由于使用async/await，因此node版本需要高于7.6 。也可以安装babel-cli，使用其中的babel-node运行。

* 中间件demo：node examples/middlewareDemo.js
* 错误处理demo： node examples/extendDemo.js
* ctx扩展demo： node examples/extendDemo.js
